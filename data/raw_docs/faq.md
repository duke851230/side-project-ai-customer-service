# 退貨政策

- 可於收到商品 7 天內申請退貨。
- 商品需保持完整包裝且未使用。
- 若為瑕疵商品，運費由平台負擔。
- 若因個人因素退貨（非瑕疵），退貨寄回運費由消費者負擔。
- 退貨申請成立後，請於 3 天內寄出退貨包裹，逾期可能自動取消申請。
- 若退貨商品有缺件、明顯使用痕跡或包裝毀損，可能酌收整新費。

# 換貨政策

- 目前僅提供「瑕疵或寄錯商品」換貨服務。
- 尺寸或款式更換請先辦理退貨，再重新下單。
- 換貨流程通常需 5-7 個工作天（含來回物流與檢驗）。
- 換貨前請保留商品外箱、配件與發票資訊，方便客服核對。

# 物流說明

- 一般配送時間約為 2-4 個工作天。
- 偏遠地區可能延長到 5-7 個工作天。
- 週末與國定假日不計入工作天，配送時效會順延。
- 預購商品將依商品頁顯示的出貨日期為準，無法與現貨保證同時到貨。
- 若同筆訂單包含預購與現貨商品，系統可能分批出貨。

# 訂單異常常見問題

- 訂單顯示「處理中」超過 48 小時：可能為付款覆核或庫存調整，建議聯繫客服查詢。
- 訂單顯示「已出貨」但物流無更新：物流資訊更新可能有 12-24 小時延遲。
- 收件資訊填錯：若尚未出貨可聯繫客服協助修改；已出貨需與物流端確認攔截可能性。
